# VBA Macro Repository

Репозиторий модульных макросов для Microsoft Excel и Microsoft Word.

Версия: **0.1.0**

## Назначение

Проект хранит VBA-код отдельно от рабочих книг и документов. Это позволяет:

- вести историю изменений в GitHub;
- повторно использовать модули в разных файлах;
- исключать дублирование имён процедур;
- тестировать отдельные функции;
- импортировать и экспортировать модули автоматически;
- постепенно собирать собственную библиотеку алгоритмов.

## Состав первой версии

### Excel

1. Форматирование фрагментов текста по разделителю.
2. Случайный выбор значений без повторов.
3. Кодирование чисел 1–54 в буквенный код и коэффициент.
4. Обратное декодирование буквенного кода.
5. Исправление опечаток по словарю и расстоянию Левенштейна.
6. OCR изображений через локальный Tesseract.
7. Самопроверка основных функций.

### Word

1. Исправление текста выделенного фрагмента по внешнему словарю.
2. Сохранение пробелов и знаков препинания.
3. Защита от объединения соседних слов.

## Быстрый запуск

1. Создайте книгу Excel в формате `.xlsm`.
2. Нажмите `Alt + F11`.
3. Выберите `File → Import File`.
4. Импортируйте модули из `src/excel`.
5. При необходимости импортируйте `tests/modSelfTests.bas`.
6. Запустите `Repository_Initialize`.
7. Запустите `Repository_SelfTest`.
8. Для OCR укажите путь к `tesseract.exe`.

Подробная инструкция находится в `docs/INSTALLATION.md`.

## Главный принцип

Каждая публичная процедура имеет уникальное имя. Поэтому ошибка:

```text
Ambiguous name detected
```

не должна возникать при однократном импорте каждого модуля.

## Безопасность

Перед запуском изучите `docs/SECURITY.md`. Не включайте доступ к объектной модели VBA без необходимости.
