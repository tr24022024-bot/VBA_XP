# Архитектура

## Уровни

### 1. Ядро

`modCore.bas` содержит общие функции:

- нормализацию текста;
- проверку существования листа;
- безопасное восстановление состояния Excel;
- сведения о версии репозитория.

### 2. Алгоритмы

Независимые функции без привязки к конкретному диапазону:

- расстояние Левенштейна;
- кодирование и декодирование;
- перемешивание массива;
- извлечение градусов, минут и секунд.

### 3. Обработчики диапазонов

Процедуры, которые получают объект `Range` или адрес диапазона.

### 4. Рабочие команды

Процедуры с окончанием `_Default` используют диапазоны, применявшиеся в текущих задачах.

### 5. Тесты

`tests/modSelfTests.bas` проверяет критические функции без изменения рабочих данных.
