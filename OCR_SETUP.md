# Локальный OCR через Tesseract

## Установка

Установите Tesseract OCR для Windows и языковые модели, необходимые для распознавания.

Типовой путь:

```text
C:\Program Files\Tesseract-OCR\tesseract.exe
```

## Языки

В процедуре можно указать:

```text
eng
rus
ukr
eng+rus
eng+rus+ukr
```

Языковые файлы должны находиться в папке `tessdata`.

## Использование

Запустите:

```vb
OCR_Image_To_ActiveSheet
```

Макрос:

1. предложит выбрать изображение;
2. запустит Tesseract локально;
3. дождётся завершения;
4. прочитает текстовый результат;
5. поместит строки в активный лист;
6. попробует извлечь градусы, минуты и секунды.

OCR не гарантирует идеальное распознавание таблиц. Поэтому исходная строка сохраняется в столбце A, а разобранные значения — в соседних столбцах.
